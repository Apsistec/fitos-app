# FitOS Mobile App

Ionic Angular application for trainers and clients.

## Setup

This app will be scaffolded using:

```bash
npx @ionic/cli start mobile blank --type=angular-standalone
```

## Development

```bash
npm start
```

## Build

```bash
npm run build
```

## Native Builds

```bash
npx cap add ios
npx cap add android
npx cap sync
```
